# CreateFolderWithSelection

在右键菜单中动态添加 “用所选项目新建文件夹“

https://github.com/MicaApps/CreateFolderWithSelection

如果你在文件资源管理器中选择了多个项目，右键菜单中会自动添加一个项目，“用所选项目新建文件夹“。如果你只选了一个项目，则不会。
